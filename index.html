Heli Escape

A terminal based game that allows you to maneuver upward out of the deep, rocky valley. Keeps score and has 23 levels. Extremely challenging and addicting, used a list of tuples. **written in Python 3**

Gladiator Game

A terminal based gladiator game that uses dictionaries, allows user to choose 1 player or 2 player. The user can choose to punch, cut, mega kick, skip, heal, or quit. Also made an Object Oriented version.

Gas Pump Program

Program that calculates the total cost for customer (gas types: Diesel, Regular, Premium, Mid-Grade) **made in Python 3 with dictionaries**

Motorcycle Rental Store

Also a Python program that uses dictionaries, allows customer to rent or return the motorcycle of their choice. **can check store revenue**

Student Checkin
